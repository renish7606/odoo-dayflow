KEY FILES DESCRIPTION:
======================

Configuration Files:
- requirements.txt          : Python package dependencies
- manage.py                 : Django project management script
- mysite/settings.py        : Django project settings (database, apps, middleware, etc.)
- mysite/urls.py            : Main URL routing configuration

Core Application (core/):
- Basic Django app structure (minimal implementation)

HRMS Application (hrms/):
- models.py                : Database models (Employee, Attendance, Leave, Payroll, etc.)
- views.py                 : View functions for handling HTTP requests
- urls.py                  : URL patterns for HRMS app
- forms.py                 : Django forms for data input
- admin.py                 : Django admin interface configuration
- signals.py               : Django signals for model events

Templates:
- base.html                : Base template with common layout
- auth/                    : Authentication pages (login, signup)
- admin/                   : Admin dashboard and management pages
- employee/                : Employee portal pages

Static Files:
- css/style.css            : Custom stylesheet
- js/main.js               : Custom JavaScript

Media Files:
- profiles/                : User-uploaded profile images

Database:
- db.sqlite3               : SQLite database file
